Ce qui donne:

\begin{enumerate}

\begin{minipage}{8.2cm}

\item $\txR f(x)=x(x-2\sqrt{3})(x+2\sqrt{3})$.
\item Un produit de facteur est nul \ssi au moins est de ses facteurs
	est nul: $$f(x)=0\Leftrightarrow x\in\{-2\sqrt{3};0;2\sqrt{3}\}$$
\item Tableau de signes ci-contre:
	$$f(x)<0\Leftrightarrow x\in]-\infty;-2\sqrt{3}[\cup]0;2\sqrt{3}[$$
		
\end{minipage}\hfill
