\maketitle
Provides \Lcs{crbox} which typesets boxes with crossing lines at corners. The syntax is:
\begin{BDef}
\Lcs{crbox}\OptArgs\Largb{\Larga{text}}
\end{BDef}
Options are \textsf{curved} and \textsf{hollowcurved}.
