The `\Boisik\kern-.25ex' font is a Metafont typeface greatly inspired by Baskerville design. It is intended for mathematical typesetting: \Boisik{} supports all characters and math symbols described in the book `\kern-.15ex{\textit{Computer Modern Typefaces}}' by \textsc{D. E. Knuth}. Symbols from $\mathbb{AMS}$ font set (\textit{msam} and \textit{msbm}) and few others are supported too. Also letters for the Czech language and text companion symbols are available.\hfill {\def\,{\kern.2ex}\color{cervena}Á\,á\,È\,è\,Ï\,ï\kern.3exÉ\,é\,Ì\,ì\,Í\,í\,Ò\,ò\,Ø\,ø\,©\,¹\,«\,»\,Ù\,ù\,Ú\,ú\,Ý\,ý\,®\,¾\,\textmho\,\textmusicalnote\,\textonehalf\,\textmu}

Current status of \Boisik{} is an alpha version: many features are missing, many don't work. I hope (\textbf{not guarantee}) in the improvement in the future. Note that the $\mathsf{Sans}$-$\mathsf{serif}$ math alphabet is faked one---\Boisik{} will never have sans-serif variant.\vskip 1mm
